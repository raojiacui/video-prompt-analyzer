# 视频反推提示词设计文档

> 版本：v2.0
> 更新时间：2026-02-21
> 用途：AI视频提示词反推系统的核心提示词设计说明

---

## 一、设计理念

### 1.1 核心目标
将视频画面**逆向工程**为可复现的AI视频生成提示词，实现：
- **准确性**：精准捕捉画面元素
- **完整性**：覆盖所有关键维度
- **可复现性**：输出提示词可直接用于AI视频生成
- **专业性**：使用影视行业标准术语

### 1.2 设计原则
| 原则 | 说明 |
|------|------|
| **分层解析** | 从粗到细，先整体后细节 |
| **结构化输出** | 信息模块化，便于机器解析 |
| **数值化描述** | 优先使用数值（如"45度"而非"斜角"） |
| **术语规范** | 采用影视行业标准术语 |
| **正向+负向** | 同时给出想要的和不想要的 |

## 二、分析框架设计

### 2.1 单帧分析框架（6维度）

```
┌─────────────────────────────────────────────────────────────┐
│                      单帧分析框架                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐   │
│  │  主体    │  │  场景    │  │  镜头    │  │  光影    │   │
│  │ Subject  │  │Environment│  │ Camera   │  │ Lighting │   │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘   │
│                                                             │
│  ┌──────────┐  ┌──────────┐                                │
│  │  风格    │  │  氛围    │                                │
│  │  Style   │  │  Mood    │                                │
│  └──────────┘  └──────────┘                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 各维度详解

#### 维度1：画面主体 (Subject)
**优先级：★★★★★**

| 子项 | 分析要点 | 示例 |
|------|----------|------|
| 人物特征 | 性别、年龄、外貌、表情、发型、妆容 | 年轻女性，长发，微笑 |
| 人物动作 | 姿态、手势、运动状态、朝向 | 行走中，朝向镜头 |
| 服装配饰 | 款式、颜色、材质、细节 | 黑色风衣，红色围巾 |
| 其他主体 | 动物/物体/图标等核心元素 | 一只金毛犬 |

#### 维度2：环境场景 (Environment)
**优先级：★★★★☆**

| 子项 | 分析要点 | 示例 |
|------|----------|------|
| 场景类型 | 室内/室外/自然/城市/虚拟 | 城市街道 |
| 具体场景 | 具体地点描述 | 涩谷十字路口 |
| 时段天气 | 时间+天气组合 | 黄昏+雨天 |
| 背景元素 | 建筑、植被、道具等 | 霓虹招牌、高楼 |
| 空间深度 | 前/中/背景层次 | 前景行人，背景建筑 |

#### 维度3：镜头语言 (Camera)
**优先级：★★★★★**

| 子项 | 选项 |
|------|------|
| **拍摄角度** | 平视 / 低角度仰拍 / 高角度俯拍 / 鸟瞰 / 虫眼 / 荷兰角 |
| **镜头景别** | 极端远景(E LS) / 远景(LS) / 全景(FS) / 中景(MS) / 近景(MCU) / 特写(CU) / 大特写(ECU) |
| **运镜方式** | 固定 / 推镜头(Dolly In) / 拉镜头(Dolly Out) / 摇摄(Pan) / 俯仰(Tilt) / 跟随 / 环绕 / 手持 |
| **焦点控制** | 合焦位置 / 景深浅深 / 焦点转移 / 背景虚化 |
| **构图法则** | 三分法 / 黄金分割 / 引导线 / 框架构图 / 对称 / 居中 / 留白 |

#### 维度4：光影照明 (Lighting)
**优先级：★★★★☆**

| 子项 | 分析要点 | 示例 |
|------|----------|------|
| 光源类型 | 自然光/人造光具体来源 | 路灯 / 窗外日光 |
| 光照方向 | 顺光/侧光/逆光/顶光/底光 | 侧逆光 |
| 光比氛围 | 柔和/强烈/高反差/低反差/剪影 | 高反差 |
| 特殊光效 | 体积光/光晕/反射/辉光 | 体积光穿透烟雾 |
| 色温色调 | 暖调/冷调/中性/赛博朋克 | 赛博朋克霓虹 |

#### 维度5：美术风格 (Style)
**优先级：★★★☆☆**

| 子项 | 分析要点 | 示例 |
|------|----------|------|
| 视觉风格 | 写实/电影/动漫/3D/油画等 | 电影感 |
| 色彩体系 | 主色+配色方案 | 蓝紫+橙红互补 |
| 饱和对比 | 饱和度和对比度描述 | 高饱和高对比 |
| 质感材质 | 表面质感描述 | 潮湿路面反光 |
| 后期处理 | 胶片颗粒/色差/晕影等 | 胶片颗粒 |

#### 维度6：氛围情绪 (Mood)
**优先级：★★★☆☆**

| 子项 | 分析要点 | 示例 |
|------|----------|------|
| 情感基调 | 整体情绪 | 孤独 / 温馨 / 神秘 |
| 叙事暗示 | 故事背景暗示 | 等待某人 |
| 感官体验 | 通感描述 | 寒冷雨夜 |
| 节奏动态 | 静止/缓慢/快速/动荡 | 缓慢流动 |

## 三、批量分析框架（序列分析）

### 3.1 序列分析特殊维度

批量分析需要额外关注：

```
┌─────────────────────────────────────────────────────────────┐
│                      序列分析框架                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │              时间连贯性 (Temporal)                     │   │
│  │   帧间变化 → 运动轨迹 → 时间流逝                       │   │
│  └─────────────────────────────────────────────────────┘   │
│                           ↓                                 │
│  ┌─────────────────────────────────────────────────────┐   │
│  │            镜头运动解析 (Camera Movement)              │   │
│  │   运镜类型 → 运动速度 → 运动轨迹 → 起止状态            │   │
│  └─────────────────────────────────────────────────────┘   │
│                           ↓                                 │
│  ┌─────────────────────────────────────────────────────┐   │
│  │            视觉一致性 (Visual Consistency)             │   │
│  │   风格统一 → 变化点 → 重复元素                         │   │
│  └─────────────────────────────────────────────────────┘   │
│                           ↓                                 │
│  ┌─────────────────────────────────────────────────────┐   │
│  │             叙事节奏 (Narrative Rhythm)                │   │
│  │   节奏类型 → 情绪曲线 → 高潮点                         │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 序列分析输出差异

| 项目 | 单帧分析 | 批量分析 |
|------|----------|----------|
| 核心描述 | 画面深度描述 | 叙事概述 |
| 镜头 | 静态镜头参数 | 运动轨迹+速度 |
| 主体 | 单帧状态 | 动作变化轨迹 |
| 输出重点 | 精准复现画面 | 捕捉动态规律 |

## 四、输出格式设计

### 4.1 输出结构

```
┌─────────────────────────────────────────────────────────────┐
│  【画面深度描述】                                            │
│  ─────────────────────────────────────────────────────────  │
│  整体性文字描述（150-200字）                                 │
├─────────────────────────────────────────────────────────────┤
│  【AI视频生成提示词】                                        │
│  ─────────────────────────────────────────────────────────  │
│  📌 核心提示词（一行版，直接可用）                           │
│  ─────────────────────────────────────────────────────────  │
│  🔸 各维度详细展开（带图标）                                 │
│  🔸 技术参数建议（宽高比、时长、负面提示词）                 │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 核心提示词设计

**设计理念**：将所有关键信息浓缩为一行，便于直接复制使用

**结构**：`[主体] + [动作] + [场景] + [镜头] + [风格]`

**示例**：
```
年轻女性雨夜东京街头行走，低角度跟拍，赛博朋克霓虹风格，电影感
```

### 4.3 图标系统

| 图标 | 用途 |
|------|------|
| 📌 | 核心提示词 |
| 🎬 | 镜头相关 |
| 👥 | 主体/人物 |
| 🏞️ | 场景环境 |
| 📷 | 镜头语言 |
| 💡 | 光影照明 |
| 🎨 | 美术风格 |
| ✨ | 氛围情绪 |
| 🔧 | 技术参数 |

## 五、优先级体系

### 5.1 维度优先级

```
主体 ★★★★★  (画面核心)
场景 ★★★★☆  (环境支撑)
镜头 ★★★★★  (视角决定)
光影 ★★★★☆  (氛围营造)
风格 ★★★☆☆  (艺术表现)
氛围 ★★★☆☆  (情感升华)
```

### 5.2 提示词优先级

当AI视频生成有限制时，按以下顺序保留：

1. **必保留**：主体、场景、镜头
2. **重要**：光影、风格
3. **可选**：氛围、细节

## 六、技术参数建议

### 6.1 宽高比推断

| 画面特征 | 推荐宽高比 |
|----------|------------|
| 电影/风景 | 16:9 / 21:9 |
| 手机竖屏内容 | 9:16 |
| 社交媒体方图 | 1:1 |
| 传统电影 | 2.35:1 |

### 6.2 运动强度分级

| 级别 | 描述 | 示例 |
|------|------|------|
| 静止 | 几乎无运动 | 固定镜头风景 |
| 微动 | 轻微运动 | 人物呼吸、微调 |
| 中等 | 正常运动 | 行走、对话 |
| 剧烈 | 快速运动 | 跑动、打斗 |

### 6.3 时长推断

| 帧间变化程度 | 建议时长 |
|--------------|----------|
| 几乎无变化 | 3-5秒（循环） |
| 轻微变化 | 5-10秒 |
| 明显变化 | 10-15秒 |
| 大幅变化 | 15秒以上 |

## 七、负面提示词设计

### 7.1 通用负面词

```
模糊, 变形, 扭曲, 低分辨率, 像素化, 伪影,
水印, 文字, 标志, logo, 边缘黑边,
画面抖动, 闪烁, 颜色失真
```

### 7.2 场景特定负面词

| 场景 | 负面词 |
|------|--------|
| 人物 | 面部扭曲, 手指异常, 眼睛不对称 |
| 动物 | 额外肢体, 形态怪异 |
| 风景 | 不自然的光线, 违和的透视 |

## 八、使用指南

### 8.1 单帧分析适用场景

- 产品展示视频参考
- 静态画面生成动态视频
- 学习优秀镜头语言

### 8.2 批量分析适用场景

- 分析完整视频片段
- 学习运镜规律
- 复刻动态效果

### 8.3 提示词使用建议

1. **核心提示词**：直接复制使用
2. **详细参数**：根据AI工具要求调整
3. **技术参数**：结合具体平台设置
4. **负面提示词**：按需添加

## 十、附录

### 10.1 术语表

| 术语 | 英文 | 说明 |
|------|------|------|
| 景别 | Shot Size | 镜头取景范围大小 |

### 10.2 构图法则速查

| 法则 | 说明 |
|------|------|
| 三分法 | 画面九宫格，主体置于交叉点 |
| 黄金分割 | 0.618比例分割 |
| 引导线 | 利用线条引导视线 |
| 框架构图 | 利用前景框住主体 |
| 对称构图 | 左右或上下对称 |
